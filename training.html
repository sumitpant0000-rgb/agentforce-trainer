Key exam traps

  • Agent User ≠ end user. The Agent User is a system user — end-user permissions don't control what the agent can access.
  • Data Library content doesn't move in change sets. Only configuration moves; data must be re-indexed in production.
  • Prompt Builder ≠ Einstein Copilot. Prompt Builder is admin-configured templates; Copilot is conversational.
  • Hybrid search ≠ always best. Match search type to data type.
  • MCP ≠ Agent API. MCP is for AI-to-AI context sharing. Agent API is for external apps talking to agents.
  • Deterministic behavior isn't the default — explicitly configure filters and variables.
  • Templates must be ACTIVATED before users can execute them.
